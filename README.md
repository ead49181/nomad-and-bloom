# 🌿 Nomad & Bloom Travel

A bohemian-inspired travel website designed and built from scratch using HTML5, CSS3, and JavaScript. The site showcases some of the world's most breathtaking destinations with a warm, wanderlust aesthetic.

## 🌍 Live Demo
[View Live Site](https://ead49181.io/nomad-and-bloom)

## ✨ Features
- Full screen parallax hero header with background image
- Fixed frosted glass navigation bar that scrolls with you
- Interactive destination grid with hover overlay descriptions
- Smooth scroll behavior throughout the page
- Background ambient music on first click
- Fully responsive layout for desktop, tablet, and mobile
- Bohemian color palette with CSS custom properties
- Custom Google Fonts typography pairing

## 🛠️ Built With
- HTML5
- CSS3 — Grid, Flexbox, CSS Variables, Media Queries, Animations
- JavaScript — Audio autoplay interaction
- Google Fonts — Playfair Display, Raleway, Cormorant Garamond
- Remixicon — Icon library
